# QA-portfolio-saucedemo
Meus estudos e práticas de testes manuais usando o site [SauceDemo](https://www.saucedemo.com/).

## Estrutura
- **plano-de-testes**: Plano completo de testes
- **test-cases/**: Casos de teste documentados
- **bug-reports/**: Relatórios de bugs encontrados
- **evidencias/**: Prints e evidências dos testes
